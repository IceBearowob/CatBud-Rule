# 申訴

## 撰寫中

## 原則

1. 申訴可以在[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)提出，提出時應符合格式
2. 申訴應在懲處後14天內提出
3. 申訴由 @行政 受理

### 申訴格式
```
Minecraft ID:
要申訴的檢舉連結:
申訴事由:
```

## 懲處判定申訴

原則上由 @行政 重審案件並全體票決決定

## 損害賠償申訴

根據申請人所損失財產，協調雙方討論，或由 @行政 執行

## 玩家領地凍結解除申訴

根據申請人提供的事由決定是否解除

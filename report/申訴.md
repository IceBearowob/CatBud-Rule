# 申訴

## 原則

1. 申訴可以在[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)提出，提出時應符合格式
2. 申訴應在懲處後14天內提出
3. 申訴由 @行政 受理
4. 申訴原則最多1次，如果案件有新的事實或證據則例外
5. 申訴期間懲處會繼續執行，如果懲處等級有更新則應根據懲處時間折抵天數

### 申訴格式
```
Minecraft ID:
要申訴的檢舉連結:
申訴類型:
申訴事由:
```

## 懲處判定申訴

由 @行政 重審案件並票決決定懲處方式與等級

## 損害賠償申訴

根據申請人所損失財產，協調雙方討論，或由 @行政 執行，詳情請看[損害賠償](https://discord.com/channels/1278590147193081947/1484750687698030692)

## 玩家領地凍結解除申訴

根據申請人與該玩家的關係、申請領地的使用比例/用途等多個要素，由 @行政 裁定是否解除凍結，詳情請看[玩家領地凍結處置方法](https://discord.com/channels/1278590147193081947/1518115442646974564)

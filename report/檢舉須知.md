# 檢舉須知

## 你應該知道的

### 關於檢舉玩家的使用
1. [檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)為唯一合法檢舉玩家的地方
> 私下檢舉不在此限，但案件仍需記錄在[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)
3. 檢舉為最終玩家糾紛處理管道，建議玩家先溝通再檢舉
4. 檢舉貼文應按照[檢舉玩家->使用方法](https://discord.com/channels/1278590147193081947/1429151949613174924)的格式貼文

### 關於懲處判定與事件處理
1. @行政 為[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)案件的主要受理、調查、懲處判定者
2. 如果 @行政 事件超過一天以上沒有進度或者特殊理由，則 @行政 有權直接進行懲處判定
3. 距離事件發生超過90天的檢舉不予受理，除特殊狀況
4. 如果並未加入貓芽DC、沒有DC、遊戲不上線等導致無法聯繫者，即自動喪失申訴或申請等相關權利

### 其他事項

* 如果需要匿名檢舉，或者不想讓被檢舉人知道是誰檢舉，可以透過私下檢舉來檢舉玩家
> 請同樣按照[檢舉玩家->使用方法](https://discord.com/channels/1278590147193081947/1429151949613174924)的格式私訊給 @行政   
> 在檢舉確定成立後，會將檢舉案件放在[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)，部分與檢舉人或證人相關資訊會消去或者匿名處理  
* 如果財產因為被檢舉人而損失，檢舉人可以申請[損害賠償](https://discord.com/channels/1278590147193081947/1484750687698030692)
* 和解會按照以下規定執行
> 如果在判決尚未確定前和解，標記為和解並終止審理   
> 如果在判決確定後和解，同樣標記為和解，但會繼續執行懲處   
> 如果該玩家經UR判決確定後和解，則將懲處下修為SSR
* 如果對懲處有疑問，或者上訴，請在[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)進行[申訴](https://discord.com/channels/1278590147193081947/1518114427940114493)
* 如果因為玩家被懲處，導致共同建築或居住的領地被凍結無法使用，可以按照[玩家領地凍結處置方法](https://discord.com/channels/1278590147193081947/1518115442646974564)在[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)進行[申訴](https://discord.com/channels/1278590147193081947/1518114427940114493)

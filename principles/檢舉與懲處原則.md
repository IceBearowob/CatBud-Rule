# 檢舉與懲處原則

## 檢舉與懲處判定原則

1. 在[詳細規章](https://discord.com/channels/1278590147193081947/1417912393438199889)內針對某些事項應規定，但無明文規定時，基於同一規範目的得引用類似規範來類推適用
2. 審判時以[規則](https://discord.com/channels/1278590147193081947/1278594306587885591)與[詳細規章](https://discord.com/channels/1278590147193081947/1417912393438199889)既有規定為主，另可參考判決先例來協助懲處判定
3. @行政 為[檢舉玩家](https://discord.com/channels/1278590147193081947/1385036894424924170)事件的受理及調查者，並由 @系統貓 執行懲處
4. 如果 @行政 告知當事人前來說明後一天，仍沒有任何訊息，則 @行政 有權直接進行懲處判定
5. 距離事件發生超過90天的檢舉不予受理

## 臨時封鎖

臨時封鎖為 @行政 於特殊狀況下的緊急處置方式  
如果被檢舉人符合下列條件，則 @行政 有權直接進行封鎖後進行案件調查及審判  
i. 脫產  
ii. 對社群造成威脅或破壞之疑慮  
iii. 調查之必要  
iv. 現行犯  
臨時封鎖的時間可以在正式懲處後抵時間

## 懲處等級

R - 遊戲內ban 1天  
SR - 遊戲內ban 3天  
SSR - 遊戲內ban 9天  
UR - 遊戲內ban 永久  

### 附加懲處

下列Discord相關懲處方式，可以與遊戲內並行懲處或單獨懲處  
禁言 - 60秒 / 5分 / 10分 / 1小時 / 1天 / 1周 ，期間不能在貓芽Discord內聊天/語音  
踢出/驅逐 - 踢出貓芽Discord  
停權 - 踢出並永久禁止加入貓芽Discord

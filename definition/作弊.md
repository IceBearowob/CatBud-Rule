# 作弊

## 常見作弊行為
### 戰鬥
* KillAura / Aura：自動攻擊周圍的敵人，幾乎不用瞄準。
* TriggerBot：準心瞄到敵人就自動攻擊。
* Reach：增加攻擊距離。
* Aimbot / BowAimbot：自動瞄準敵人（近戰或遠程）。
* Criticals：強制每次攻擊都判定為暴擊。
### 移動
* Fly / Flight：能像創造模式一樣飛行。
* Speed：移動速度超快。
* Step / Spider：能像爬牆或一次跨過好幾格方塊。
* NoFall：免疫摔落傷害。
* Jesus：能在水面上行走。
### 生存
* FastPlace / FastBreak：加快放方塊或挖掘速度。
* Inventory Hacks：修改背包物品、無限物品。
* Tweakroo-停用方塊挖掘冷卻：將客戶端0.3秒的挖掘冷卻移除
* X-ray：透視"未裸露"的礦物及箱子
* Litematica Printer：藍圖打印機
### 其他特殊
* 自動BOT：就是機器人
## 不違反規則的功能
* Litematica-EasyPlace 快速放置功能
* InvMove
> 請注意:禁止在PVP中使用
